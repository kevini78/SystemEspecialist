# Sistema Especialista de Sintomas

Sistema especialista para diagnóstico diferencial de sintomas.

Sistema é utilizado para conseguir ajudar a destinguir doenças, de acordo com o sintomas, e esse quadro de sintomas foi criado por um médico especialista.

Passos para rodar o sistema:

#1
Ter node.js baixado

#2 Baixar as dependências via: pip install requirements.txt

#3 executar no terminal cd health-compass e após isso, rodar npm run dev
